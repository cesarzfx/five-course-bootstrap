# five-course-bootstrap

This project is a single-page website created as part of a class assignment focused on building a responsive webpage using Bootstrap.

The site presents an **Italian Dinner at Home** theme and demonstrates the use of **Bootstrap containers, rows, columns, responsive utilities, and cards** to organize a five-course meal layout with images and summarized recipe descriptions.

---

## Page Overview
- **Home Page**:  
  Includes a responsive hero section, an introductory menu description area, and a structured five-course meal layout featuring:
  - Welcome Drink
  - Appetizer
  - Salad Course
  - Main Course
  - Dessert

Each meal section includes an image, title, course label, and a short recipe summary.

## Assignment Objectives
This project demonstrates:
- Creating a responsive single-page layout using Bootstrap
- Using a Bootstrap container with multiple rows and columns
- Organizing content with Bootstrap’s grid system
- Building a five-course meal webpage with images and summarized recipes
- Applying responsive layout changes across multiple screen sizes
- Combining Bootstrap classes with custom CSS for styling and presentation
